conheça a fisioterEPIC!

A fisioterEPIC! Foi um projeto realizado durante o primeiro semestre do curso de análise e desenvolvimento de sistemas. Nesse projeto, o objetivo era criar um site funcional para uma clínica médica. Após uma breve reunião decidimos fazer uma página para uma clínica de fisioterapia fictícia. Essa versão do site foi desenvolvida apenas por mim com o objetivo de testar minhas habilidades em CSS e Bootstrap.
